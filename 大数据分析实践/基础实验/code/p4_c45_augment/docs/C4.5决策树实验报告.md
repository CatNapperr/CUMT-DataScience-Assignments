## 2. 实验目标和实验环境

### 2.1 实验目标

本实验旨在利用 C4.5 决策树算法，根据 NBA 球员的赛季技术统计特征预测其场上位置。C4.5 在 ID3 的基础上引入三项核心改进：**增益率（Gain Ratio）替代信息增益**、**连续特征自动分裂**、**悲观后剪枝（Pessimistic Pruning）**。

实验设两个子任务：
1. **5 位置细粒度分类**：预测具体位置（C, PF, SF, SG, PG）
2. **3 超类粗粒度分类**：合并为 Big（C+PF）、Wing（SF+SG）、PG 三个超类

同时对比无剪枝和有剪枝两种方案，并与原始 C4.5 实验（场均统计 + KBinsDiscretizer）进行跨版本对比。

## 3. 算法原理

### 3.1 C4.5 的改进

C4.5（Quinlan, 1993）是 ID3 的演进版本，主要改进如下：

#### （1）增益率（Gain Ratio）

针对 ID3 信息增益偏好取值多的特征的问题，C4.5 使用增益率：

$$GainRatio(D, A) = \frac{Gain(D, A)}{SplitInfo(D, A)}$$

其中分裂信息 SplitInfo 为：

$$SplitInfo(D, A) = -\sum_{i=1}^{n} \frac{|D_i|}{|D|} \log_2 \frac{|D_i|}{|D|}$$

增益率通过除以分裂信息来惩罚取值多的特征，使分裂更均衡。

#### （2）连续特征自动分裂

C4.5 原生支持连续特征：将样本按特征值排序后，在相邻不同类别的样本之间取中点作为候选阈值，选择增益率最大的阈值进行二分。

#### （3）悲观后剪枝

C4.5 采用悲观剪枝策略控制过拟合：对每个非叶子节点，比较将其替换为叶子节点后的误差估计（含连续性校正 +0.5）与原子树误差估计，如果替换更优则剪枝。

## 4. 数据预处理与特征工程

### 4.1 Per-36 分钟归一化

消除出场时间影响，将所有累计统计转换为每 36 分钟尺度：

$$StatPer36 = \frac{Stat}{MP} \times 36$$

### 4.2 TOT 去重与样本过滤

- 赛季中转会球员：保留 TOT 行，删除单球队行
- 保留 MP ≥ 200 分钟且 G ≥ 10 场的球员

预处理后共 13,273 条有效样本。

### 4.3 特征体系

本实验采用 **15 维特征**，分为基础统计和风格特征两类：

#### 基础统计特征（9 项）

| 特征 | 含义 |
|------|------|
| PTS_per36 | 每 36 分钟得分 |
| AST_per36 | 每 36 分钟助攻 |
| TRB_per36 | 每 36 分钟篮板 |
| STL_per36 | 每 36 分钟抢断 |
| BLK_per36 | 每 36 分钟盖帽 |
| FG% | 投篮命中率 |
| FT% | 罚球命中率 |
| PF_per36 | 每 36 分钟犯规 |
| TOV_per36 | 每 36 分钟失误 |

#### 风格特征（6 项）

| 特征 | 含义 | 公式 |
|------|------|------|
| Has3P | 是否有三分出手 | 3PA > 0 |
| HasFT | 是否有罚球记录 | FTA > 0 |
| 3PA_per36 | 每 36 分钟三分出手 | 3PA/MP×36 |
| AST/TOV | 助攻失误比 | AST/TOV |
| FTr | 罚球率（造犯规倾向） | FTA/FGA |
| 3PAr | 三分出手占比（外线倾向） | 3PA/FGA |

相比原始 C4.5，本实验删除了 3P%（与 3PA_per36、3PAr 信息重叠）和 STL_BLK（综合防守指标），新增了 3PAr（三分出手占比），使特征更加聚焦于**打法倾向**而非单纯的数量统计。

## 6. 与原始 C4.5 实验对比

### 6.1 预处理与特征差异

| 维度 | 原始 C4.5 | 新 C4.5 |
|------|----------|---------|
| 统计口径 | 场均（Per Game） | **Per 36 分钟** |
| 样本过滤 | G ≥ 20，删除 TOT 行 | **MP ≥ 200 + G ≥ 10，TOT 去重** |
| 样本量 | 13,861 | **13,273** |
| 连续值处理 | **KBinsDiscretizer（5 箱等频）** | **C4.5 原生连续分裂** |
| 特征维度 | 16 维 | **15 维** |
| 关键特征 | 含 3P%、STL_BLK | **含 3PAr**，删除 3P% |

### 6.2 5 位置分类对比

| 指标 | 原始 C4.5（有剪枝） | 新 C4.5（有剪枝） | 变化 |
|------|:-----------------:|:----------------:|:----:|
| 测试集准确率 | 53.50% | **54.55%** | +1.05% |
| 过拟合差距 | 5.31% | **0.48%** | -4.83% |
| 叶子节点数 | 193 | **31** | -83.9% |
| Macro F1 | **0.5294** | 0.4792 | -0.0502 |
| 每叶样本数 | 50.28 | **299.71** | +496% |

5 位置分类中，准确率小幅提升，但 Macro F1 有所下降，主要原因是连续分裂没有 KBinsDiscretizer 的平滑效果，对 SF 这类边界模糊位置的泛化更弱。但树极度精简（193→31 叶子），过拟合几乎完全消除。

### 6.3 3 超类分类对比

| 指标 | 原始 C4.5（有剪枝） | 新 C4.5（有剪枝） | 变化 |
|------|:-----------------:|:----------------:|:----:|
| 测试集准确率 | 76.10% | **82.60%** | **+6.50%** |
| 过拟合差距 | 3.23% | **0.61%** | -2.62% |
| 叶子节点数 | 126 | **32** | -74.6% |
| Macro F1 | 0.7542 | **0.8195** | +0.0653 |
| 每叶样本数 | 77.02 | **290.34** | +277% |

**3 超类分类提升显著**（+6.5%），per-36 归一化 + 风格特征 + 连续分裂的综合效果在此完全体现。更关键的是，树从 126 叶子压缩到 32 叶子，每叶样本数从 77 提升到 290，决策规则更加稳定可靠。

### 6.4 提升原因分析

1. **Per-36 归一化消除出场时间偏差**：使助攻、篮板等统计在相同尺度下比较，更能反映球员真实技术特点而非出场机会
2. **TOT 去重消除数据泄漏**：避免多球队球员被重复计数导致的训练偏差
3. **风格特征增强区分能力**：3PAr、FTr、AST/TOV 等比率特征直接刻画打法倾向，比单纯的数量统计更具位置判别力
4. **连续分裂保留完整信息**：摒弃 KBinsDiscretizer 离散化，C4.5 原生连续分裂不损失信息，找到的最优阈值更加精细
5. **悲观剪枝有效控制过拟合**：原版 C4.5 仍有 3-5% 过拟合差距，新版本降至 0.5-0.6%，接近完全消除

## 8. 决策树结构（3 超类，有剪枝）

C4.5 生成的决策树以 TRB_per36（篮板）为根节点，这是区分内线（Big）与外线（Wing/PG）最有效的特征。根节点以下按 AST_per36（助攻）划分控卫与侧翼。完整树共 32 个叶子节点，生成 32 条分类规则。

部分关键规则示例：

```
IF TRB_per36 ≤ 6.69 AND AST_per36 ≤ 4.87 AND ... THEN Pos = Big
IF TRB_per36 ≤ 6.69 AND AST_per36 > 4.87 AND STL_per36 > 0.45 AND ... THEN Pos = PG
IF TRB_per36 > 6.69 AND AST_per36 ≤ 10.53 AND ... THEN Pos = Big
```

这些规则直观反映了篮球位置的判别逻辑：篮板高 → 内线；助攻多 + 抢断多 → 控卫；助攻少 + 三分多 → 侧翼。

## 9. 实验结论

本实验实现了基于 C4.5 决策树的 NBA 球员位置分类，采用 per-36 归一化、TOT 去重和 15 维特征（9 基础 + 6 风格），得到以下结论：

1. **C4.5 有效克服 ID3 过拟合**：通过增益率 + 连续分裂 + 悲观剪枝，将过拟合差距从 ID3 的 41.63% 压缩至 0.61%

2. **3 超类分类效果显著**：测试准确率 82.60%，Macro F1 0.8195，与 Naive Bayes 持平且具有可解释性优势

3. **5 位置分类仍具挑战**：测试准确率 54.55%，PF/SF 边界模糊问题无法通过算法或特征工程完全解决

4. **per-36 + 风格特征贡献明显**：相比原始 C4.5，3 超类准确率提升 6.5%，树压缩 74.6%，验证了特征优化的有效性

5. **剪枝几乎不影响准确率**：剪枝后准确率仅下降 0.02-0.07%，但叶子数减少约 50%，显著提升可解释性
